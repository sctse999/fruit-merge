# Grand Prix — Pseudo-3D F1 Racer

Browser-based Out Run–style pseudo-3D racing game built with **Three.js** and **Vite**.

## Run locally

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (usually `http://localhost:5173`).

## Controls

- **W / ↑** — Accelerate
- **S / ↓** — Brake
- **A / D** or **← / →** — Steer
- **Space** — Start / restart

## Build

```bash
npm run build
npm run preview
```
